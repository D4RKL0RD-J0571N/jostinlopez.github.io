## 🧠 Design Philosophy

> _“Systems are stories told in logic.”_

I believe gameplay systems are creative expressions — not just code.  
They shape how players **feel** and **interact** with the world.  

My approach is built around four principles:

- 🔹 **Expandability:** Build once, extend forever.  
- 🔹 **Simplicity:** Clean structure beats clever tricks.  
- 🔹 **Empowerment:** Tools should feel like superpowers.  
- 🔹 **Immersion through logic:** When systems blend, the world breathes.
